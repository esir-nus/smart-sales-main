# 03 对话

> 你的 AI 销售助手 — 随时提问、分析、跟进

---

## 对话是什么

对话界面是你和 SmartSales AI 助手沟通的主要方式。你可以：

- 直接打字提问（如"帮我整理今天拜访的要点"）
- 绑定录音后针对会议内容提问
- 让 AI 帮你安排跟进任务
- 获取客户信息的分析和建议

---

## 消息类型

### 用户消息

```
            ┌─────────────────────┐
            │  你输入的消息内容     │
            └─────────────────────┘
                       右对齐，深色气泡
```

- 发送后立即显示（乐观渲染）
- 右对齐，便于区分

### AI 回复

```
┌──────────────────────────────┐
│  AI 回复的内容                │
│  支持 Markdown 格式：         │
│  • 列表项                    │
│  • **加粗** 和 *斜体*        │
│  • 代码块等                  │
└──────────────────────────────┘
  左对齐，浅色气泡
```

- 左对齐，区别于用户消息
- 支持富文本格式渲染（列表、加粗、表格等）
- 长回复支持流式输出，逐步显示

### AI 活动状态条

当 AI 正在处理你的请求时，会显示一个两层状态条：

```
┌──────────────────────────────┐
│ 阶段: 📝 规划分析步骤         │  ← 当前高级任务
│ 动作: 🧠 思考中...            │  ← 具体操作
├──────────────────────────────┤
│ 正在分析录音转写文本，提取    │  ← 思考过程
│ 关键讨论点和行动项...         │     （可展开）
└──────────────────────────────┘
```

**阶段类型：**

| 阶段 | 含义 |
|------|------|
| 📝 规划 (PLANNING) | 正在分析你的问题，制定回答策略 |
| ⚡ 执行 (EXECUTING) | 正在检索信息或处理数据 |
| 💬 回复 (RESPONDING) | 正在生成回答 |
| ❌ 出错 (ERROR) | 遇到问题，会告诉你原因 |

**动作类型：**

| 动作 | 含义 |
|------|------|
| 🧠 思考中 | AI 正在推理 |
| 📄 解析中 | 正在处理文档或转写文本 |
| 🎙 转写中 | 正在转写录音 |
| 🔍 检索中 | 正在查找相关信息 |
| 🔧 组装中 | 正在整合多个信息源 |
| 📡 输出中 | 正在生成最终回复 |

> **活动状态条会保留在对话历史中**，你可以回溯查看 AI 是如何得出结论的。

## 绑定录音上下文

这是 SmartSales 最强大的功能之一 — 把录音"绑定"到对话中，让 AI 基于真实的会议内容回答你的问题。

### 方法一：从录音抽屉发起

1. 打开录音抽屉（从底部上滑）
2. 找到已转写的录音
3. 点击 **「Ask AI」** 按钮
4. 自动回到对话界面，录音已绑定

```
录音抽屉中：
┌────────────────────────────────┐
│  张总拜访_20260416             │
│  时长: 45:32  |  已转写 ✓     │
│                   [Ask AI]     │  ← 点击这里
└────────────────────────────────┘
         ↓
对话界面中：
┌────────────────────────────────┐
│ 🎵 已绑定: 张总拜访_20260416   │  ← 上下文指示条
└────────────────────────────────┘
```

### 方法二：从对话中附加

1. 在对话界面点击输入框左侧的 📎 附件按钮
2. 打开录音抽屉（自动进入选择模式）
3. 选择一段录音
4. 返回对话，录音已绑定

```
点击 📎 后的录音抽屉（选择模式）：
┌────────────────────────────────┐
│  选择录音                      │
├────────────────────────────────┤
│  ● 张总拜访_20260416  [已转写] │  ← 可选择
│  ○ 李经理电话_20260415 [已转写]│  ← 可选择
│  ○ 周会录音_20260414   [转写中]│  ← 可选择（会立即绑定）
│                                │
│  ◌ 当前已绑定的录音    [禁用]  │  ← 已绑定的不可重选
└────────────────────────────────┘
```

### 切换录音上下文

在对话过程中，你可以随时切换绑定的录音：

1. 再次点击 📎
2. 选择另一段录音
3. 上下文自动切换

> **注意**：切换录音不会清除之前的对话记录，但 AI 后续的回答会基于新绑定的录音。

### 正在转写的录音

如果你绑定了一段还在转写中的录音：
- 绑定会立即生效
- 对话中会显示转写进度
- 转写完成后，AI 可以正常回答相关问题
- 不会出现虚假的确定性 — 如果内容还没准备好，AI 会明确告诉你

## 对话的特性

### 基于真实信息

SmartSales AI 的设计原则是 **基于事实回答**：

- 基于你的录音转写、日程数据回答问题
- 不会编造没有根据的信息
- 如果不确定，会明确告诉你而不是胡编
- 所有建议都可以追溯到源数据

### 上下文记忆

在同一个会话中：
- AI 记得你之前说的内容
- 可以进行多轮追问
- 绑定的录音信息持续可用

### 任务衔接

从对话中可以直接衔接到日程：
- AI 可能建议创建跟进任务
- 你确认后，任务会出现在日程抽屉中
- 不会静默创建任何内容 — 总是需要你确认

## 实操示例

### 示例 1：分析会议录音

```
你: 帮我整理今天和张总会议的要点

AI: [阶段: 📝 规划] [动作: 📄 解析转写文本]
    
    根据你的会议录音，以下是主要讨论要点：

    **客户需求：**
    • 张总对当前方案的报价基本满意
    • 希望增加售后服务条款
    • 关注交付时间是否能提前

    **行动项：**
    • 周三前发送修改后的合同
    • 确认技术团队的交付排期
    • 准备售后服务补充协议

    需要我帮你安排这些跟进任务吗？
```

### 示例 2：提取关键信息

```
你: 张总提到了哪些竞品？

AI: 在录音的 12:34 和 28:15 处，张总提到了两家竞品：
    
    1. **XX科技** — 价格比我们低 10%，但张总认为
       他们的技术方案不够完善
    2. **YY系统** — 张总之前用过，服务体验一般

    张总对我们的整体评价偏正面，主要顾虑是交付时间。
```

### 示例 3：请求跟进建议

```
你: 会后我应该怎么跟进？

AI: 根据会议内容，我建议以下跟进步骤：

    **紧急（本周内）：**
    1. 发送修改后的合同（张总说周三前需要）
    2. 和技术部确认交付排期

    **一般（下周）：**
    3. 准备售后服务补充方案
    4. 安排技术演示（张总提到想看实际效果）

    需要我帮你把这些添加到日程中吗？
```

## 常见问题

**Q: AI 回复太慢怎么办？**
长录音的分析可能需要一些时间。活动状态条会告诉你当前进度。

**Q: AI 的回答不准确？**
AI 基于转写文本工作。如果转写有误，可以在录音抽屉中查看原始转写并重试。

**Q: 可以不绑定录音直接聊天吗？**
可以。SmartSales 支持普通对话，不绑定录音也能聊天、提问、安排日程。

**Q: 一个会话能绑定多段录音吗？**
每次只能绑定一段录音，但可以随时通过 📎 切换。之前的对话内容会保留。
