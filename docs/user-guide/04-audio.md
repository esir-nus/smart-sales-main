# 04 录音管理

> 从录音到洞察 — 录音抽屉完全指南

---

## 什么是录音抽屉

录音抽屉是你管理所有录音文件的中心。它从屏幕底部滑出，包含录音列表、转写状态、工件查看等功能。

**打开方式：**
- 从主界面底部 **向上滑动**（28dp 的触发区域）
- 在对话中点击 **📎 附件按钮**（进入选择模式）

### 录音卡片状态

每条录音都以卡片形式呈现，有三种状态：

#### 待转写

```
┌─────────────────────────────────┐
│  客户电话_20260413              │
│  8:45  |  待转写                │
│                → 右滑触发转写    │
└─────────────────────────────────┘
```

- 录音已同步但尚未处理
- **右滑** 可触发转写

#### 转写中

```
┌─────────────────────────────────┐
│  周会录音_20260414              │
│  1:23:05  |  转写中 ⏳           │
│  ████████░░ 72%                 │
└─────────────────────────────────┘
```

- 显示转写进度条
- 进度实时更新
- 无需等待 — 可以继续其他操作

#### 已转写

```
┌─────────────────────────────────┐
│ ★ 张总拜访_20260416             │
│   45:32  |  已转写 ✓            │
│   摘要 | 章节 | 关键词           │
│                      [Ask AI]   │
└─────────────────────────────────┘
```

- 显示可查看的工件标签（摘要、章节、关键词等）
- **★ 星标** 表示已收藏
- **Ask AI** 一键跳转到对话，绑定此录音

| 工件 | 说明 | 用途 |
|------|------|------|
| 📝 **转写全文** | 带时间戳的逐字稿 | 精确回顾每个细节 |
| 📋 **摘要** | 整体内容的精炼概括 | 快速了解会议要点 |
| 📖 **章节** | 按话题自动分段 | 跳转到感兴趣的部分 |
| 🔑 **关键词** | 提取的核心话题标签 | 快速把握讨论范围 |
| 🎤 **说话人** | 发言人识别和占比 | 了解谈话节奏 |
| ❓ **问答** | 提取的问答对 | 梳理待回答的问题 |

> **工件内容是基于转写文本加工的**，不会编造原文中没有的信息。显示文本可能做过可读性优化（如修正明显的语音识别错误），但不会改变原意。

## 卡片操作

### 滑动手势

```
  ← 左滑：删除录音
┌──────────────────────┐
│  录音名称             │  ← → 右滑：触发转写
└──────────────────────┘      （仅未转写状态）
```

| 手势 | 操作 | 适用状态 |
|------|------|----------|
| → 右滑 | 触发转写 | 待转写 |
| ← 左滑 | 删除录音 | 所有状态 |

### 星标收藏

- 点击录音卡片上的 **★/☆** 切换收藏状态
- 收藏的录音在列表中有视觉标记
- 用于标记重要录音方便后续查找

### Ask AI

- 点击已转写录音的 **「Ask AI」** 按钮
- 关闭录音抽屉，回到对话界面
- 录音自动绑定到当前对话

## 手动同步

```
┌──────────────────────────────────┐
│  录音管理            [🔄 手动同步] │
└──────────────────────────────────┘
                         ↑ 点击这里
```

- **位置**：录音抽屉顶部右侧的胶囊按钮
- **功能**：触发与录音徽章的同步，获取最新录音
- **何时使用**：
  - 录音徽章刚结束录制
  - 怀疑有未同步的录音
  - 重新连接设备后

> **同步不会自动打开录音抽屉** — 你需要手动触发，不是后台自动运行。

## 选择模式

当你从对话中通过 📎 按钮打开录音抽屉时，进入 **选择模式** — 一个简化的界面，专注于选择录音：

```
┌────────────────────────────────────┐
│  选择录音                           │
├────────────────────────────────────┤
│                                    │
│  ◌ 当前绑定: 张总拜访_20260416     │  ← 灰色禁用
│                                    │
│  ○ 李经理电话_20260415  [已转写]   │  ← 可选择
│  ○ 周会录音_20260414    [转写中]   │  ← 可选择
│  ○ 客户电话_20260413    [待转写]   │  ← 可选择
│                                    │
└────────────────────────────────────┘
```

与浏览模式的区别：
- **无教学提示**（省去引导文字）
- **无同步胶囊**（专注选择）
- **当前绑定的录音** 显示为禁用状态
- 选择后 **立即返回对话**，录音自动绑定

## 录音来源

### 主要来源：录音徽章

- 通过蓝牙配对的 SmartSales 录音徽章
- 录制完成后通过 Wi-Fi 上传
- 手动同步触发获取

### 测试用途：本地导入

- 用于开发和测试场景
- 可从手机存储中选择音频文件
- 支持常见音频格式

> **录音转写服务由通义听悟（Tingwu）提供**，需要网络连接。转写结果持久化保存，已转写的录音不会重复处理。

## 新手引导

如果你是第一次使用录音抽屉，且只有示例录音（demo seed），系统会显示简短的引导提示帮助你了解操作方式。这些提示在你添加真实录音后会自动消失。

## 常见问题

**Q: 转写需要多长时间？**
取决于录音长度和网络状况。通常 10 分钟的录音需要 1-3 分钟转写。

**Q: 转写失败了怎么办？**
录音抽屉会显示错误状态和原因。常见原因包括网络中断或服务暂时不可用。你可以重新触发转写。

**Q: 可以删除转写结果但保留录音吗？**
目前删除会同时移除录音和转写结果。

**Q: 转写支持哪些语言？**
主要支持中文，也支持中英混合的会议场景。

**Q: 录音文件会一直保存在手机上吗？**
录音文件和转写结果保存在本地。可以在设置中查看存储占用并清理。
