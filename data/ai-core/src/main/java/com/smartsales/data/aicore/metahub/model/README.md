# MetaHub Model Layer

**Spec Reference:** Orchestrator-V1.md §4

## Purpose

DTOs for each Metadata Hub layer.

## Models

- `UserMetadata.kt` - M1: language, style, tone, preferences
- `ConversationDerived.kt` - M2: tags, highlights, action items
- `TranscriptionDerived.kt` - M2B: chapters, keyPoints, speakers
- `SessionState.kt` - M3: session name, links, visibility
