# 02 主界面

> 你的工作基地 — 所有功能的出发点

---

## 界面总览

主界面（Home Shell）是 SmartSales 的核心框架。所有功能都从这里触达。

```
┌─────────────────────────────────────┐
│ ☰    📶    [ 动态岛 ]          [+]  │  ← 顶栏（固定不动）
├─────────────────────────────────────┤
│                                     │
│                                     │
│          ┌───────────────┐          │
│          │   中央画布     │          │
│          │               │          │
│          │  （对话内容    │          │
│          │   或欢迎页）   │          │
│          │               │          │
│          └───────────────┘          │
│                                     │
│                                     │
├─────────────────────────────────────┤
│ [📎]  输入消息...            [发送]  │  ← 底部输入框（固定不动）
└─────────────────────────────────────┘
         ▲ 极光背景底色 ▲
```

### 三层结构

主界面由三个固定层组成，无论你在做什么操作，顶栏和底部输入框始终可见：

| 层 | 位置 | 说明 |
|----|------|------|
| **顶栏** | 顶部 | 导航入口和状态显示，贯穿全屏 |
| **中央画布** | 中间 | 内容区域，显示对话或欢迎页 |
| **输入框** | 底部 | 消息输入和附件操作，贯穿全屏 |

## 顶栏详解

```
┌──────────────────────────────────────────┐
│  ☰     📶      [下一个: 拜访张总 14:00]  [+]  │
│  ①     ②              ③                  ④   │
└──────────────────────────────────────────┘
```

### ① 菜单按钮 ☰

- **位置**：左上角
- **点击**：打开 [历史记录抽屉](06-history.md)
- **功能**：查看和切换过去的会话

### ② 设备徽章 📶

- **位置**：左侧，菜单按钮旁
- **功能**：显示录音徽章的连接状态
- **点击**：打开 [设备连接弹窗](07-connectivity.md)

**状态指示：**

| 状态 | 表现 | 含义 |
|------|------|------|
| 已连接 | 稳定显示，每 30 秒脉冲 5 秒 | 徽章工作正常 |
| 重连中 | 常驻显示在动态岛 | 正在尝试重新连接 |
| 离线 | 每 30 秒脉冲 2.5 秒 | 徽章未连接 |
| 需要设置 | 常驻显示在动态岛 | 尚未配对任何设备 |

### ③ 动态岛

- **位置**：顶栏居中
- **功能**：单行状态摘要，显示最重要的信息
- **点击**：打开 [日程抽屉](05-scheduler.md)

**动态岛会显示什么：**

| 优先级 | 内容 | 示例 |
|--------|------|------|
| 默认 | 下一个日程任务 | `下一个: 拜访张总 14:00` |
| 日程冲突 | 冲突提醒 | `⚠ 14:00 有两个安排冲突` |
| 连接异常 | 设备状态接管 | `📶 重新连接中...` |

> **交互方式**：点击动态岛打开日程抽屉。如果动态岛当前显示设备状态，点击则打开设备连接弹窗。

### ④ 新建会话 +

- **位置**：右上角
- **点击**：开始一个全新的对话会话
- **效果**：当前对话保存到历史，界面回到欢迎状态

## 中央画布

中央画布是主界面的核心内容区域，有两种状态：

### 空会话状态（欢迎页）

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│                                 │
│     你好, SmartSales 用户        │
│     我是您的销售助手             │
│                                 │
│                                 │
│                                 │
│                                 │
└─────────────────────────────────┘
```

- 显示问候语和副标题
- 发送第一条消息后，欢迎页消失，进入对话视图
- 底部有旋转的提示语（shimmer hint），提供使用灵感

### 对话状态

```
┌─────────────────────────────────┐
│                                 │
│  ┌──────────────────────┐       │
│  │ AI: 好的，我来帮你分析 │      │  ← AI 回复（左对齐）
│  │ 这段录音的要点...     │       │
│  └──────────────────────┘       │
│                                 │
│       ┌──────────────────────┐  │
│       │ 你: 帮我总结刚才会议 │   │  ← 用户消息（右对齐）
│       └──────────────────────┘  │
│                                 │
│  ┌──────────────────────┐       │
│  │ 阶段: 📝 规划分析步骤 │       │  ← AI 活动状态
│  │ 动作: 🧠 思考中...    │       │
│  └──────────────────────┘       │
│                                 │
└─────────────────────────────────┘
```

详细说明见 [03 对话](03-chat.md)。

## 底部输入框

```
┌──────────────────────────────────────┐
│  [📎]  在这里输入消息...       [发送] │
│   ①              ②              ③    │
└──────────────────────────────────────┘
```

### ① 附件按钮 📎

- 点击打开录音抽屉，选择录音绑定到当前对话
- 也可用于打开文件选择器

### ② 消息输入区

- 单行起始，输入文字后自动增长
- 最多扩展至 4 行（高度上限 140dp）
- 支持多行输入

### ③ 发送按钮

- 点击发送消息
- 输入内容为空时不可点击

## 手势导航

SmartSales 支持直觉化的手势操作，不需要记菜单位置：

```
                  ↓ 从动态岛向下拉
                  打开日程抽屉
                  
    ← 从左侧滑入          从右侧滑入 →
    打开历史抽屉          （系统返回）

                  ↑ 从底部向上拉
                  打开录音抽屉
```

| 手势 | 起始位置 | 打开 |
|------|----------|------|
| ↓ 向下拉 | 动态岛 / 顶部 | 日程抽屉 |
| ↑ 向上拉 | 底部边缘（28dp 区域） | 录音抽屉 |
| ← 从左侧滑入 | 屏幕左边缘 | 历史抽屉 |

### 抽屉互斥规则

**同一时间只能打开一个抽屉。** 如果你打开了录音抽屉，再尝试打开日程抽屉，录音抽屉会先自动关闭。

关闭抽屉的方式：
- 点击抽屉外的半透明遮罩
- 使用抽屉底部的关闭手柄
- 系统返回键

## 视觉层级

SmartSales 的界面有清晰的层级关系。当你打开抽屉或弹窗时，背后的内容会被半透明遮罩（scrim）覆盖，帮助你聚焦当前操作：

```
最上层   │  系统弹窗（确认框、提示）
  ▲     │  抽屉手柄（录音/日程的常驻触发区）
  │     │  抽屉面板（录音、日程、历史）
  │     │  半透明遮罩（40% 黑色）
  │     │  浮动元素（徽章、按钮）
  │     │  内容区域（对话气泡、列表）
最底层   │  背景（极光底色）
```

---

## 视觉风格

- **深色优先**：默认深色主题，视觉安静、专业
- **极光背景**：主界面底部有微妙的极光渐变效果
- **毛玻璃材质**：卡片和面板使用半透明毛玻璃质感
- **色彩克制**：以黑色为基底，蓝色为强调色，减少视觉噪音
