## 2. 软件简介

### 2.1 一句话总述
Smart Sales Assistant（智能销售助手）是一款专注于智能销售管理的综合信息平台，旨在通过AI技术提供全面的销售对话监测、智能分析和个性化的销售策略建议。

### 2.2 设计背景

#### 为何开发此软件？
在B2B和B2C销售场景中，销售人员面临以下核心挑战：
- **对话记录困难**：传统手工记录客户需求容易遗漏关键信息，影响后续跟进质量
- **客户洞察提取效率低**：从大量对话中提取客户痛点、购买意向等结构化信息耗时耗力
- **销售策略缺乏数据支撑**：难以基于历史对话数据形成系统化的销售策略建议
- **多设备数据同步不便**：硬件录音设备与移动应用之间的数据同步和协作流程复杂

#### 解决什么问题？
本软件通过以下方式解决上述问题：
- 提供硬件录音设备与Android应用的无线连接和数据同步能力，实现销售对话的自动记录
- 集成AI语音转写技术，将录音自动转换为文字，支持说话人分离
- 利用大语言模型对转写内容进行智能分析，自动提取客户需求、痛点、购买意向等关键信息
- 提供结构化的数据导出功能，支持PDF报告和CSV格式，便于导入CRM系统（如Salesforce、HubSpot）
- 提供实时AI聊天助手，支持多轮对话和上下文理解，辅助销售人员制定销售策略

### 2.3 目标用户
- **B2B销售代表**：面向企业客户的销售人员，需要记录和分析复杂的商务谈判过程
- **B2C销售代表**：面向个人消费者的销售人员，需要快速捕捉客户需求和购买意向
- **客户经理**：负责维护长期客户关系，需要基于历史对话数据制定个性化服务策略
- **业务拓展人员**：需要系统化记录和分析潜在客户信息，优化销售流程

### 2.4 主要功能概览

本软件提供以下核心功能（不展开操作细节）：

1. **AI智能聊天助手**
   - 支持多轮对话和上下文理解
   - 提供销售分析、客户洞察、策略建议等技能模式
   - 支持Markdown格式的响应展示和导出

2. **音频转写与分析**
   - 支持从硬件设备同步音频文件
   - 集成阿里云通义听悟（Tingwu）服务，实现高精度语音转写
   - 支持说话人分离（Diarization），区分销售人员和客户对话
   - 自动生成转写摘要，便于快速浏览

3. **设备连接与配网**
   - 支持BLE（低功耗蓝牙）设备扫描和配对
   - 提供WiFi配网功能，将移动设备连接的WiFi凭据安全传输至硬件设备
   - 支持设备状态监控和网络诊断
   - 提供步骤化的设备设置向导界面

4. **媒体文件同步与管理**
   - 支持从硬件设备同步图像、视频、音频等媒体文件
   - 提供媒体文件的上传、下载、删除、应用等操作
   - 支持音频文件的本地播放控制

5. **会话历史管理**
   - 支持查看和管理历史聊天会话
   - 提供会话搜索和时间分组功能
   - 支持会话数据的持久化存储

6. **用户中心**
   - 提供用户资料管理功能
   - 支持应用设置和偏好配置

### 2.5 技术优势与创新点

- **AI算法集成**：深度集成阿里云DashScope大语言模型和通义听悟语音转写服务，提供高精度的自然语言理解和语音识别能力
- **跨平台兼容**：基于Android平台开发，支持Android 8.0（API 26）及以上版本，兼容主流Android设备
- **实时同步**：通过BLE和WiFi双通道实现硬件设备与移动应用的实时数据同步，支持增量同步和断点续传
- **模块化架构**：采用MVVM架构模式和模块化设计，功能模块独立可测试，便于维护和扩展
- **现代化UI**：基于Jetpack Compose和Material 3设计规范，提供流畅的用户体验和响应式界面
- **离线能力**：支持Fake实现模式，在无网络或API密钥缺失时仍可进行功能演示和测试

## 3. 运行环境与技术架构

### 3.1 硬件环境

#### 最低配置要求
- **设备类型**：Android智能手机或平板电脑
- **操作系统**：Android 8.0（API Level 26）及以上版本
- **内存**：建议2GB RAM以上
- **存储空间**：建议预留100MB以上可用存储空间（用于应用安装和缓存）
- **网络**：支持WiFi和移动数据网络（用于AI服务和云端同步）
- **蓝牙**：支持BLE 4.0及以上版本（用于硬件设备连接）

#### 推荐配置
- **操作系统**：Android 10.0（API Level 29）及以上版本
- **内存**：4GB RAM以上
- **存储空间**：500MB以上可用存储空间
- **网络**：稳定的WiFi连接（用于大文件同步和AI服务调用）

#### 支持的设备
- 支持所有符合Android 8.0及以上版本的Android设备
- 已测试设备包括：主流品牌智能手机（华为、小米、OPPO、vivo、三星等）和Android平板电脑
- 硬件设备需支持BLE 4.0及以上版本，推荐使用Nordic UART Service（NUS）协议的设备（如BT311）

## 4. 系统功能详述

### 4.1 首页/聊天模块

#### 功能模块名称
首页（Home）和智能聊天助手（AI Chat Assistant）

#### 界面布局说明
- **顶部导航栏**：显示应用标题和主要功能入口
- **主内容区**：
  - 聊天消息列表：采用LazyColumn实现，支持消息气泡展示（用户消息和AI助手消息采用不同样式）
  - 消息输入框：位于界面底部，支持文本输入和发送按钮
  - 技能选择按钮：提供"分析"、"导出PDF"、"导出CSV"等快捷技能切换
  - 滚动到底部按钮：当消息列表较长时，显示浮动按钮，点击可快速滚动至最新消息
- **操作按钮**：
  - 发送按钮：提交用户输入的消息
  - 复制Markdown按钮：将当前会话的Markdown格式内容复制到剪贴板
  - 导出按钮：支持导出为PDF或CSV格式

#### 核心功能描述
该页面/模块能够实现以下功能：

1. **AI智能对话**
   - 用户输入文本消息后，系统调用阿里云DashScope大语言模型（Qwen系列）生成AI回复
   - 支持流式响应（Streaming），实时显示AI生成的内容
   - 支持多轮对话，保持上下文理解能力
   - 支持技能标签（Skill Tags），如"分析"、"导出PDF"等，影响AI回复的风格和内容

2. **消息管理**
   - 自动保存用户消息和AI回复到本地数据库（Room）
   - 支持消息时间戳显示
   - 支持Markdown格式的消息内容渲染（如标题、列表、代码块等）
   - 支持消息的复制和分享功能

3. **转写内容导入**
   - 支持将音频转写结果（Markdown格式）导入到聊天会话中
   - 转写内容可作为上下文信息，辅助AI进行客户分析和策略建议

4. **导出功能**
   - 支持将当前会话导出为PDF格式，包含完整的对话内容和格式
   - 支持导出为CSV格式，便于导入CRM系统（如Salesforce、HubSpot）
   - 导出文件保存在应用缓存目录，可通过系统分享功能分享给其他应用

#### 操作流程说明
用户如何使用该模块：

1. **开始对话**
   - 打开应用，进入首页
   - 在底部输入框输入问题或需求（如"分析一下刚才的客户对话"）
   - 点击发送按钮或按回车键提交消息

2. **查看AI回复**
   - 系统自动调用AI服务，实时显示生成的内容
   - 支持Markdown格式的富文本展示（标题、列表、代码块等）
   - 消息自动滚动至最新位置

3. **使用技能功能**
   - 点击顶部的技能选择按钮（如"分析"、"导出PDF"）
   - 技能标签会影响AI回复的风格和内容
   - 可以同时选择多个技能标签

4. **复制和导出**
   - 点击"复制Markdown"按钮，将当前会话的Markdown内容复制到剪贴板
   - 点击"导出"按钮，选择PDF或CSV格式，系统生成文件后提示用户分享

5. **查看历史会话**
   - 通过导航进入"会话历史"页面
   - 选择历史会话，可继续之前的对话

#### 管理员特有功能（如有）
本模块无管理员特有功能，所有功能面向普通用户开放。

### 4.2 设备管理模块

#### 功能模块名称
设备管理（Device Manager）和WiFi/BLE测试器（WiFi & BLE Tester）

#### 界面布局说明
- **设备扫描区域**：
  - 扫描按钮：启动/停止BLE设备扫描
  - 设备列表：显示扫描到的BLE设备，包括设备名称、信号强度（RSSI）、设备地址等信息
  - 扫描状态指示器：显示当前扫描状态（扫描中/已停止）
- **WiFi配置区域**：
  - WiFi名称输入框：输入目标WiFi网络的SSID
  - WiFi密码输入框：输入WiFi密码（支持密码隐藏显示）
  - 发送凭据按钮：将WiFi凭据通过BLE发送给硬件设备
- **网络状态查询区域**：
  - 查询网络状态按钮：查询设备当前网络连接状态
  - 网络信息显示：显示设备IP地址、连接状态等信息
- **媒体服务器管理区域**：
  - 服务地址输入框：输入硬件设备媒体服务器的base URL（如`http://192.168.0.109:8000`）
  - 刷新按钮：刷新媒体文件列表
  - 上传按钮：上传本地文件到设备
  - 媒体文件网格：以网格形式展示设备上的图像、视频、音频文件

#### 核心功能描述
该页面/模块能够实现以下功能：

1. **BLE设备扫描与连接**
   - 扫描附近的BLE设备，支持按设备名称关键词和Service UUID过滤
   - 显示设备列表，包括设备名称、MAC地址、信号强度等信息
   - 支持设备选择和连接，建立GATT连接
   - 自动发现设备的GATT服务和特征（Characteristic），优先选择支持WRITE+NOTIFY或WRITE+READ的特征

2. **WiFi配网**
   - 将移动设备当前连接的WiFi凭据（SSID和密码）通过BLE安全传输给硬件设备
   - 支持手动输入WiFi名称和密码
   - 支持热点凭据获取（当设备需要连接手机热点时）
   - 显示配网状态和结果反馈

3. **网络诊断**
   - 查询设备当前网络连接状态
   - 获取设备IP地址和网络配置信息
   - 支持打开设备Web控制台（如设备提供Web管理界面）
   - 支持打开媒体管理器（跳转到设备管理页面）

4. **媒体文件管理**
   - 从硬件设备获取媒体文件列表（图像、视频、音频）
   - 支持文件上传：将本地文件上传到设备
   - 支持文件下载：将设备文件下载到本地
   - 支持文件删除：删除设备上的文件
   - 支持文件应用：将文件应用到设备显示（如图片显示在设备屏幕上）
   - 支持视频/音频文件的预览和播放控制

#### 操作流程说明
用户如何使用该模块：

1. **扫描和连接设备**
   - 打开"WiFi & BLE Tester"页面
   - 点击"开始扫描"按钮，系统开始扫描附近的BLE设备
   - 在设备列表中选择目标设备（如BT311）
   - 系统自动建立BLE连接

2. **配置WiFi网络**
   - 在WiFi配置区域，输入WiFi名称（SSID）和密码
   - 或点击"获取热点凭据"按钮，获取手机热点信息
   - 点击"发送WiFi凭据"按钮，系统通过BLE将凭据发送给设备
   - 等待设备连接WiFi，系统显示连接状态反馈

3. **查询网络状态**
   - 点击"查询网络状态"按钮，系统向设备发送网络查询请求
   - 查看返回的网络信息（IP地址、连接状态等）
   - 如设备提供Web控制台，可点击"打开控制台"按钮在浏览器中打开

4. **管理媒体文件**
   - 在"服务地址"输入框中输入设备的媒体服务器地址（如`http://192.168.0.109:8000`）
   - 点击"刷新"按钮，获取设备上的媒体文件列表
   - 点击文件卡片上的"下载"按钮，将文件下载到本地
   - 点击"上传"按钮，选择本地文件上传到设备
   - 点击"删除"按钮，删除设备上的文件
   - 点击"应用"按钮，将文件应用到设备显示

#### 管理员特有功能（如有）
本模块无管理员特有功能，所有功能面向普通用户开放。

### 4.3 音频文件模块

#### 功能模块名称
音频文件管理（Audio Files）和音频转写（Audio Transcription）

#### 界面布局说明
- **同步状态区域**：
  - 同步按钮：触发从设备同步音频文件
  - 状态指示器：显示当前同步状态（空闲/同步中/转写中/已完成/错误）
  - 进度条：显示同步或转写进度
- **音频文件列表**：
  - 文件列表：以列表形式展示已同步的音频文件
  - 文件信息：显示文件名、文件大小、修改时间等
  - 转写状态：显示每个文件的转写状态（未转写/转写中/已完成）
- **转写摘要区域**：
  - 转写摘要显示：以Markdown格式显示转写结果摘要
  - 说话人分离信息：区分销售人员和客户的对话内容
- **播放控制区域**：
  - 播放/暂停按钮：控制音频播放
  - 播放进度条：显示播放进度
  - 播放时间显示：显示当前播放时间和总时长

#### 核心功能描述
该页面/模块能够实现以下功能：

1. **音频文件同步**
   - 从硬件设备的媒体服务器获取音频文件列表
   - 支持增量同步，只下载新增或修改的文件
   - 将音频文件下载到本地缓存目录
   - 显示同步进度和状态

2. **音频转写**
   - 将音频文件上传到阿里云OSS（对象存储服务）
   - 调用通义听悟（Tingwu）服务提交转写任务
   - 支持说话人分离（Diarization），自动区分不同说话人的对话内容
   - 实时轮询转写任务状态，显示转写进度
   - 转写完成后，生成Markdown格式的转写摘要

3. **转写结果管理**
   - 查看转写摘要，支持Markdown格式渲染
   - 支持将转写结果导入到聊天会话，作为上下文信息
   - 支持转写结果的复制和分享

4. **音频播放控制**
   - 支持本地音频文件的播放和暂停
   - 显示播放进度和播放时间
   - 支持播放状态管理（播放中/已暂停/已停止）

#### 操作流程说明
用户如何使用该模块：

1. **同步音频文件**
   - 打开"音频库"页面
   - 确保设备已连接并配置了正确的媒体服务器地址
   - 点击"同步"按钮，系统开始从设备同步音频文件
   - 查看同步进度条，等待同步完成

2. **转写音频文件**
   - 同步完成后，在音频文件列表中选择需要转写的文件
   - 系统自动执行以下步骤：
     a. 将音频文件上传到阿里云OSS
     b. 调用通义听悟服务提交转写任务
     c. 实时轮询转写任务状态
   - 转写完成后，在转写摘要区域查看转写结果
   - 转写结果支持Markdown格式，包含说话人分离信息

3. **使用转写结果**
   - 点击"导入到聊天"按钮（如有），将转写结果导入到聊天会话
   - 在聊天页面中，转写内容可作为上下文，辅助AI进行客户分析
   - 或直接复制转写摘要，分享给其他应用

4. **播放音频文件**
   - 在音频文件列表中，点击文件项
   - 使用播放控制按钮（播放/暂停）控制音频播放
   - 查看播放进度条和播放时间

#### 管理员特有功能（如有）
本模块无管理员特有功能，所有功能面向普通用户开放。

### 4.4 设备设置模块

#### 功能模块名称
设备设置向导（Device Setup Wizard）

#### 界面布局说明
- **步骤指示器**：显示当前设置步骤和总步骤数（如"步骤1/3"）
- **步骤内容区域**：
  - 步骤1：设备扫描和选择
  - 步骤2：WiFi配置
  - 步骤3：连接确认
- **操作按钮**：
  - "下一步"按钮：进入下一个设置步骤
  - "上一步"按钮：返回上一个设置步骤
  - "完成"按钮：完成设置流程
  - "取消"按钮：取消设置流程

#### 核心功能描述
该页面/模块能够实现以下功能：

1. **步骤化设备设置**
   - 将设备连接和配网流程分解为多个步骤，引导用户逐步完成
   - 每个步骤提供清晰的说明和操作指引
   - 支持步骤间的前进和后退导航

2. **设备扫描和选择**
   - 在第一步中，自动启动BLE设备扫描
   - 显示扫描到的设备列表，支持设备选择
   - 验证设备连接状态

3. **WiFi配置**
   - 在第二步中，提供WiFi配置界面
   - 支持自动获取当前WiFi信息或手动输入
   - 将WiFi凭据发送给设备

4. **连接确认**
   - 在最后一步中，验证设备连接和网络状态
   - 显示设置完成提示
   - 支持跳转到设备管理页面

#### 操作流程说明
用户如何使用该模块：

1. **启动设置向导**
   - 从首页或设备管理页面进入"设备连接"页面
   - 系统自动进入第一步：设备扫描

2. **扫描和选择设备**
   - 系统自动开始扫描附近的BLE设备
   - 在设备列表中选择目标设备
   - 点击"下一步"按钮，进入WiFi配置步骤

3. **配置WiFi**
   - 在WiFi配置界面，输入WiFi名称和密码
   - 或使用"自动获取"功能，获取当前连接的WiFi信息
   - 点击"发送凭据"按钮，将WiFi信息发送给设备
   - 等待设备连接WiFi成功
   - 点击"下一步"按钮，进入连接确认步骤

4. **完成设置**
   - 在连接确认步骤，系统验证设备连接状态
   - 显示设置完成提示
   - 点击"完成"按钮，跳转到设备管理页面，开始使用设备

#### 管理员特有功能（如有）
本模块无管理员特有功能，所有功能面向普通用户开放。

#### 功能模块名称
会话历史（Chat History）

#### 界面布局说明

- **会话列表**：
  - 按时间分组显示会话（如"今天"、"本周"、"本月"等）
  - 每个会话项显示会话标题、最后更新时间、消息数量等信息
  - 支持会话删除功能（长按或滑动删除）
- **操作按钮**：
  - 新建会话按钮：创建新的聊天会话
  - 删除按钮：删除选中的会话

#### 核心功能描述
该页面/模块能够实现以下功能：

1. **会话列表展示**
   - 从本地数据库（Room）加载历史会话列表
   - 按时间分组显示（今天、本周、本月、更早）
   - 显示每个会话的最后更新时间、消息数量等摘要信息

2. **会话搜索**
   - 支持实时搜索功能，根据关键词过滤会话
   - 搜索范围包括会话标题和消息内容

   - 支持创建新会话
   - 支持删除会话（包括会话中的所有消息）
   - 支持选择会话，跳转到聊天页面继续对话

4. **会话持久化**
   - 所有会话数据保存在本地Room数据库中
   - 应用重启后，会话数据自动恢复

   - 系统自动加载并显示所有历史会话
   - 会话按时间分组，便于快速查找

2. **搜索会话**
   - 在顶部搜索栏输入关键词
   - 系统实时过滤匹配的会话
   - 清除搜索内容，恢复完整列表

3. **继续对话**
   - 点击会话项，跳转到聊天页面
   - 系统自动加载该会话的历史消息
   - 可以继续之前的对话

4. **管理会话**
   - 长按会话项，显示删除选项
   - 或使用滑动删除手势，删除不需要的会话
   - 点击"新建会话"按钮，创建新的聊天会话
