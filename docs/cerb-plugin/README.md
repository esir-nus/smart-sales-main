# Cerb Plugins Ecosystem

This directory contains the specifications and architectural rules for **System III: The Hands**.

Plugins are independent, decoupled tools that the Unified Pipeline (System II) can route to. They interact with the core ecosystem *only* through the `PluginGateway` protocol.
