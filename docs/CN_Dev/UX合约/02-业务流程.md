## 1. 引导流程 (Onboarding Flow)

> **状态**: ✅ 已发布
> **概述**: 高保真的“开箱”旅程。重点关注硬件可靠性、账户绑定和建立信心。

### 黄金主线 (12 阶段)

1.  **Welcome**: 品牌英雄屏 ("您的 AI 销售教练")。无登录按钮，纯净入口。
2.  **Permissions**: 原生弹窗前的玻璃卡片预热 (麦克风, 蓝牙)。
3.  **Voice Handshake**: 通过手机麦克风进行"初识"。
4.  **Hardware Wake**: 引导用户长按设备按钮激活 (蓝灯闪烁)。
5.  **Scan (Radar)**: 高保真脉冲雷达动画。
6.  **Found (Manual Select)**: **禁止自动连接**。用户必须点击卡片确认。
7.  **WiFi Setup**: 通过蓝牙配置网络 (SSID/密码)。
8.  **FW Check & Update**: 强制更新固件 (< v1.2.0)。阻塞流程。
9.  **Device Naming**: 个性化命名。
10. **Account Gate**: 绑定设备以保存设置。支持登录/注册。
11. **Profile Collection**: 收集姓名、角色、行业 (可跳过)。
12. **Complete**: 成功动画 -> 跳转主页。

## 2. 日程管理流程 (Scheduler Flows)

> **状态**: ✅ 已发布
> **概述**: 语音转写 -> 意图识别 -> 日程创建/修改。

### 核心场景

#### 语音指令创建
用户录音 "帮我约张总下周二..." -> Tingwu 转写 -> 意图识别 -> 日程抽屉自动打开 -> 此时卡片状态为 `[待确认]`。
用户点击 `[Confirm]` -> 任务激活 (设置闹钟)。

#### 语音修正 (MODIFY Intent)
用户录音 "不是三点，是四点" -> 识别修正意图 -> 查找最近 (<5min) 的待确认任务 -> 更新字段 -> 检查冲突。
如果有冲突，卡片显示 `⚠️ [Conflict]` 和解决方案选项。

#### 冲突解决
1.  **检测**: 时间重叠。
2.  **通知**: 卡片角标 + Toast + (如果是Badge录音) 震动反馈。
3.  **解决**: 展开冲突卡片 -> 选择保留 A 或 B (Rethink流程)。

#### 智能提醒 (Smart Cascade)
如果没有指定提醒时间，Agent 根据任务类型自动推断：
- **会议**: 提前30分钟 (准备) -> 提前5分钟 (入会)。
- **截止日**: 提前1小时 -> 提前15分钟。
- **个人**: 提前10分钟。
> **学习**: Agent 会观察用户习惯 (是否频繁推迟第一个提醒)。

## 3. 教练模式流程 (Coach Flows)

> **状态**: ✅ 已发布
> **概述**: 快速、对话式的即时指导。

- **Simple Chat**: 基础问答。
- **Memory Search**: 当用户问题模糊 ("那个价格问题怎么样了？")，LLM 识别歧义 -> 触发后台记忆搜索 -> UI 显示 Thinking Box 提示。
- **Suggest Analyst Switch**: 当用户提出复杂数据问题，Coach 识别"需要深度分析" -> 在回复后追加 `[💡 建议切换到分析师模式]` 按钮。

## 4. 分析师模式流程 (Analyst Flows)

> **状态**: ✅ 已发布
> **概述**: 结构化、深度的规划分析。

### 完整分析 (Plan Card)
1.  用户输入 "分析张总的采购周期"。
2.  **Thinking Box** 激活 (流式显示思考过程)。
3.  生成 **分析计划卡片 (Plan Card)**，渲染在聊天流中。
    *   包含步骤: 1. 概况总结 [完成], 2. 周期计算 [等待], 3. 策略生成 [等待]。
4.  **AI 提议**: "计划已生成。是否继续进行周期计算？" (口头提议，而非单纯按钮)。
5.  用户确认 "Go ahead" -> 工具执行下一步 -> 卡片更新。

## 5. 音频与实体流程 (Audio & Entity Flows)

> **状态**: ✅ 已发布
> **概述**: 录音管理与后台实体学习。

### 录音管理
- **本地上传**: 音频抽屉 `[+]` -> 系统选择器 -> 自动转写。
- **Badge 同步**: 下拉音频抽屉 -> 触发同步 -> 新文件出现。

### 实体识别 (Entity Recognition)
- **Snakebar 通知**: 实时分析时，底部出现 `✨ 习得新实体: "Project A3"`。后台更新 Relevancy Library。

### 歧义消解 (Disambiguation)
当用户提到 "张总" 且有多个匹配时：
1.  **高置信度 (Auto-Resolved)**: 自动选择最高分候选项，并显示 "我理解您指的是张总(A3)... [更改]"。
2.  **低置信度/手动更改 (Picker)**: 弹出选择器，列出所有候选项 (按分数排序)，支持 "⭐ 常联系" 标记。选择后强化该偏好。
