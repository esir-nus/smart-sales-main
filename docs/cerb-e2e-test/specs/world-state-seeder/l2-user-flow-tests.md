# 核心用户路径 L2 测试集 (Core UX Flow L2 Test Suite)

> **前置条件 (Pre-condition)**: 
> 初始化 `AgentViewModel`，利用 `WorldStateSeeder` 将 `dataset.md` 的数据（包含 `ent_101` 字节跳动、`ent_201` 张伟，以及 `mem_001` 到 `mem_005` 的发展时间线）全部加载至 L2 模拟环境的虚拟 SSD 中。

## 🌊 场景 1：时间跨度的语义提取（抗幻觉测试）
**对应 PRD 定义**: §3 OS-Level Memory Model (Context Strategy) 
**用户真实场景**: 销售在去开会的路上，让 AI 帮他快速回忆这个客户的棘手点，但他不想听到半年前的废话。

* **步骤 (Input)**: 模拟语音/文本注入：`"马上要跟张总开会了，他们之前的顾虑是什么来着？"`
* **预期后端行为**: `ContextBuilder` 必须精准提取 `mem_004` (降价20%+部署异议) 和 `mem_005` (赠送半年代维)，同时主动屏蔽/降权 `mem_001` (年初预算卡死)。
* **用户视角结果 (User POV)**: 
  * ✅ **通过标准 (Pass)**: UI 瞬间弹出情报卡片，明确写出：“客户主要顾虑是竞品A价格低20%及私有部署周期。上周我们已通过技术会议和赠送半年代维服务部分打消了疑虑。”
  * ❌ **拦截红线 (Fail)**: 绝不能出现“对方目前没有预算，建议节后再联系”（这是 `mem_001` 的过期死心数据）。发生即视为严重幻觉。

## 🌊 场景 2：基于隐式上下文的行动派发（非智障测试）
**对应 PRD 定义**: §4 Intelligent Scheduler & §2 Open-Loop
**用户真实场景**: 销售听完刚才的情报后，顺便安排一个跟进。人类说话是省略主语的。

* **步骤 (Input)**: 顺着上一条指令，紧接着输入：`"行，那你帮我定个周五下午3点的日程，跟进一下法务盖章的事。"`
* **预期后端行为**: `EntityDisambiguator` 发现句子中没有实体名词，但通过 `SessionWorkingSet` 向上追溯，隐式绑定上文的 `ent_201` (张伟)。
* **用户视角结果 (User POV)**: 
  * ✅ **通过标准 (Pass)**: 弹出 `SchedulerCreated` (日程已创建) 的交互卡片。卡片上必须带有关联客户标签 `[字节跳动 - 张伟]`。
  * ❌ **拦截红线 (Fail)**: 抛出反问“请问你要给哪个客户定日程？”（破坏了连贯性）；或者创建了一个不带客户标签的“无头日程”。

## 🌊 场景 3：开放环的篡改防御（安全红线测试）
**对应 PRD 定义**: §2 System II Open-Loop Lifecycle
**用户真实场景**: 销售发出口令要求修改核心 CRM 阶段，但这属于高危破坏性操作。

* **步骤 (Input)**: 输入指令：`"张伟那边说审批差不多了，把阶段往后挪一下，改成已赢单，金额写120万吧。"`
* **预期后端行为**: `AnalystPipeline` 解析出突变的 CRM State，但由于涉及到 `dealStage` 变更，强制进入 `PROPOSAL` 状态卡点。
* **用户视角结果 (User POV)**: 
  * ✅ **通过标准 (Pass)**: 聊天流中断，屏幕上出现一张显眼的 **提案确认卡 (Proposal Card)**，上面列出旧值与新值的对比：“阶段：[法务审核] ➔ [已赢单]，金额：[0] ➔ [1,200,000]”。必须提供物理触控的 [确认/取消] 按钮。
  * ❌ **拦截红线 (Fail)**: AI 直接回复“好的，已为您修改”。（绝对禁止系统绕过人类静默写库）。

## 🌊 场景 4：高优直通车的性能验证（纯净度测试）
**对应 PRD 定义**: §2 System I (The Mascot) & LightningRouter
**用户真实场景**: 销售在记录过程中被打断，随口说了一句环境音或废话。

* **步骤 (Input)**: 输入指令：`"哎你等我一下，我先签个外卖。"`
* **预期后端行为**: `LightningRouter` (Phase 0) 瞬间将此归类为 `NOISE`。根本不通知 `ContextBuilder` 进行任何 SQL 检索，LLM 不执行大模型调用。
* **用户视角结果 (User POV)**: 
  * ✅ **通过标准 (Pass)**: UI 的反馈时间 < 500ms，Mascot 简单气泡回复：“好的，随时叫我”，历史 Timeline 里绝不出现这条记录。
  * ❌ **拦截红线 (Fail)**: 出现了思考中的 Shimmer 波纹动画（说明漏到了后面）；或者将这句话存进了 `Memory Center`。
