## 场景一：客户拜访全流程

从拜访前准备到拜访后跟进的完整操作流程。

### 拜访前

```
时间线：出发前 30 分钟

步骤 1：查看今日日程
├── 点击动态岛，打开日程抽屉
├── 确认拜访时间和地点
└── 查看是否有冲突

步骤 2：准备录音设备
├── 确认徽章已充电
├── 检查 📶 设备状态为「已连接」
└── 确认 Wi-Fi 配置正确（特别是换了办公环境后）

步骤 3（可选）：回顾上次沟通
├── 打开历史抽屉 ☰
├── 找到上次与该客户的会话
└── 浏览之前的分析要点和待办事项
```

### 拜访中

```
步骤：佩戴徽章，开始录音

  ┌─────────────────────────────────┐
  │                                 │
  │  徽章正在录音                    │
  │  你只需要专注谈话               │
  │  录音在徽章本地存储              │
  │                                 │
  └─────────────────────────────────┘

提示：
• 不需要看手机，专注与客户交流
• 徽章独立录制，不依赖手机连接
• 会议结束后再处理录音
```

### 拜访后

```
时间线：会议结束后

步骤 1：同步录音
├── 打开 SmartSales
├── 从底部上滑打开录音抽屉
├── 点击 [🔄 手动同步]
└── 等待录音传输完成

步骤 2：转写录音
├── 找到刚同步的录音卡片
├── 右滑触发转写
└── 等待转写完成（可以先做别的）

步骤 3：查看分析
├── 转写完成后，展开录音卡片
├── 先看「摘要」快速了解要点
├── 再看「章节」找到关键段落
└── 「问答」查看需要跟进的问题

步骤 4：AI 深度分析
├── 点击录音卡片的 [Ask AI]
├── 问："帮我整理这次会议的要点和行动项"
├── 追问："客户提到的竞品有哪些？"
└── 追问："我应该怎么跟进？"

步骤 5：创建跟进任务
├── 在对话中："帮我根据会议安排跟进任务"
├── AI 列出建议的任务和时间
├── 确认后任务自动创建到日程抽屉
└── 打开日程抽屉确认任务已就位
```

**完整流程示意：**

```
拜访前            拜访中            拜访后
                  
查看日程    →    佩戴徽章    →    同步录音
检查设备         专注交流         触发转写
回顾历史                         AI 分析
                                 安排跟进
```

## 场景二：每日工作开始

```
步骤 1：打开 SmartSales
├── 查看动态岛的今日概要
└── 有冲突？点击动态岛处理

步骤 2：回顾待办
├── 打开日程抽屉
├── 浏览今天的任务列表
├── 有紧急任务（L1/L2）优先处理
└── 有冲突卡片需要调整

步骤 3：处理昨日遗留
├── 打开历史抽屉
├── 检查未完成的会话
└── 继续昨天的跟进

示意图：
┌─────────────────────────────────┐
│  早上打开 APP                    │
│       ↓                         │
│  动态岛 → 有冲突？              │
│  ├ 是 → 打开日程处理            │
│  └ 否 → 查看今日任务            │
│       ↓                         │
│  处理最紧急的待办                │
│       ↓                         │
│  继续昨天未完成的对话            │
└─────────────────────────────────┘
```

## 场景三：会议录音批量处理

当你一天有多个会议，可以这样批量处理：

```
步骤 1：统一同步
├── 打开录音抽屉
├── 点击 [🔄 手动同步]
└── 所有新录音一次性出现

步骤 2：批量触发转写
├── 逐个右滑未转写的录音
├── 转写并行处理，不需要等一个完成再开始下一个
└── 可以关闭录音抽屉，转写在后台继续

步骤 3：逐个分析
├── 为每段录音创建独立会话（点击 + 新建）
├── 绑定录音 → Ask AI → 提问分析
├── 用有意义的名字给会话重命名
└── 为每段录音安排各自的跟进任务

提示：
┌─────────────────────────────────────┐
│ 效率技巧：                           │
│                                     │
│ • 先全部同步和触发转写               │
│ • 利用转写等待时间处理其他工作        │
│ • 转写完成后按重要性排序分析          │
│ • 每个客户一个会话，方便后续查找      │
└─────────────────────────────────────┘
```

## 场景四：准备客户周报

```
步骤 1：收集本周拜访信息
├── 打开历史抽屉
├── 找到本周所有客户相关的会话
└── 逐个回顾 AI 的分析总结

步骤 2：让 AI 帮你汇总
├── 新建一个会话
├── "帮我整理本周所有客户拜访的要点"
├── 逐个绑定本周的录音
└── 请 AI 按客户/主题分类总结

步骤 3：提取行动项
├── "本周有哪些待跟进的事项？"
├── "哪些客户需要优先关注？"
└── 将重要跟进安排到下周日程

提示：给每周的汇总会话起个好名字
（如"W16 周报整理"），方便后续查找。
```

## 效率技巧

### 手势速查

| 想要做什么 | 最快的方式 |
|------------|-----------|
| 查看/管理日程 | 点击动态岛 |
| 浏览/管理录音 | 从底部上滑 |
| 切换历史会话 | 点击 ☰ 或从左边缘右滑 |
| 新建对话 | 点击右上角 + |
| 查看设备状态 | 点击 📶 |
| 关闭任何抽屉 | 点击灰色遮罩区域 |

### 命名规范建议

给会话起有意义的名字，方便后续查找：

| 场景 | 推荐命名 | 不推荐 |
|------|----------|--------|
| 客户拜访 | "张总-Q2方案讨论-0416" | "帮我分析录音" |
| 周报整理 | "W16 周报整理" | "新对话" |
| 项目跟进 | "XX项目-合同修改" | "录音分析" |

### 录音管理建议

- **及时同步**：会议结束后尽快同步，避免积压
- **及时转写**：同步后立即触发转写，利用等待时间做别的
- **星标重要录音**：用 ★ 标记关键客户的录音
- **定期清理**：在设置中检查存储占用，清理不需要的旧录音

### 日程管理建议

- **每天早上检查**：打开日程抽屉回顾今天的安排
- **及时标完成**：完成任务后立即打勾，保持清单准确
- **处理冲突**：不要忽略冲突卡片，及时调整
- **用 AI 批量创建**：会后让 AI 一次性帮你安排所有跟进
