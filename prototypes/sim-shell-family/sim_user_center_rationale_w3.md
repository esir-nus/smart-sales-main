# Wave 3 Rationale

The row system is engineered for a premium, calm reading experience:
- **Anatomy**: `height: 52px` rows with clear icon-label-action flow. Primary text is bright (`text-primary`), metadata values and chevrons are muted (`text-secondary`), maintaining a clear hierarchy without shouting.
- **Card Treatment**: Borderless rows inside a dark frosted container with ultra-thin `0.5px` dividers (`rgba(255,255,255,0.05)`), keeping the layout light and sophisticated. 
- **Icons & Toggles**: Standard minimalist stroke SVGs and refined OS-style toggles to denote state intuitively.
- **Logout Constraint**: The destructive action uses pure red text (`#FF453A`) on a nearly transparent dark base (`rgba(255,69,58,0.08)`). It sits completely outside the card system at the very bottom, avoiding accidental taps while communicating its gravity clearly.
