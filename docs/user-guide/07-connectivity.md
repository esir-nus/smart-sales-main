# 07 设备连接

> 录音徽章配对、状态监控和故障排查

---

## 录音徽章是什么

SmartSales 录音徽章是一个独立的硬件设备，佩戴在身上可以：
- 高质量录制会议和拜访
- 通过蓝牙与手机连接
- 通过 Wi-Fi 自动上传录音

---

## 连接状态

### 设备徽章指示

主界面左上方的 📶 图标实时显示连接状态：

```
┌──────────────────────────────────┐
│ ☰  📶  [ 动态岛 ]           [+]  │
│      ↑                           │
│   设备徽章                        │
└──────────────────────────────────┘
```

| 状态 | 徽章表现 | 动态岛 | 含义 |
|------|----------|--------|------|
| ✅ 已连接 | 稳定显示，每 30 秒脉冲 5 秒 | 显示日程信息 | 一切正常 |
| 🔄 重连中 | 显示中 | 显示「重连中...」 | 正在恢复连接 |
| ⭕ 离线 | 每 30 秒脉冲 2.5 秒 | 显示日程信息 | 未连接设备 |
| ⚙️ 需要设置 | 显示中 | 显示「需要设置」 | 从未配对 |

### 查看连接详情

点击 **📶 设备徽章** 打开连接弹窗：

## 首次配对

如果你在初始设置中跳过了设备配对，可以随时配对：

### 进入配对流程

1. 点击 📶 设备徽章
2. 弹窗显示「需要设置」
3. 点击 **「配对新设备」**

## Wi-Fi 配置

录音徽章通过 Wi-Fi 上传录音到云端。如果你换了办公场所或 Wi-Fi 密码变了，需要更新配置。

### 修改 Wi-Fi

1. 从动态岛或连接弹窗进入连接详情
2. 点击 **「修改 Wi-Fi」**
3. 输入新的网络名称和密码
4. 等待配置完成

### 同步失败后的自动提示

如果你点击同步录音时，手机当前 Wi-Fi 和徽章所在 Wi-Fi 不一致，应用现在会直接弹出已有的 Wi-Fi 配置窗口，不需要你再先点一次连接弹窗。

- 表单会优先带入手机当前连接的 Wi-Fi 名称
- 你仍然可以手动修改网络名称
- 输入密码并确认后，应用会把当前网络配置发给徽章

```
┌─────────────────────────────────┐
│  修改 Wi-Fi 配置                 │
│                                 │
│  当前网络: MyOffice_5G          │
│                                 │
│  新网络名称: [____________]     │
│  密码:       [____________]     │
│                                 │
│  [取消]              [保存]     │
└─────────────────────────────────┘
```

## 连接状态在动态岛中的接管

当设备连接异常时，动态岛会临时从日程显示切换为连接状态：

```
正常状态：
┌───────────────────────────────────┐
│ ☰  📶  [下一个: 拜访张总 14:00]  [+] │
└───────────────────────────────────┘

连接异常时：
┌───────────────────────────────────┐
│ ☰  📶  [📶 重新连接中...]        [+] │
└───────────────────────────────────┘
```

- 连接恢复后自动切回日程显示
- 点击动态岛此时打开的是连接弹窗而非日程抽屉

## 故障排查

### 徽章无法被发现

| 检查项 | 操作 |
|--------|------|
| 徽章是否开机 | 长按 3 秒，确认蓝灯闪烁 |
| 蓝牙是否开启 | 检查手机蓝牙设置 |
| 距离是否太远 | 将徽章靠近手机（1 米以内） |
| 电量是否充足 | 给徽章充电后重试 |

### 连接频繁断开

| 检查项 | 操作 |
|--------|------|
| 距离问题 | 保持手机和徽章在合理范围内 |
| 干扰问题 | 远离其他蓝牙设备密集的区域 |
| 电量低 | 检查徽章电量 |

### 录音上传失败

| 检查项 | 操作 |
|--------|------|
| Wi-Fi 是否连接 | 检查徽章的 Wi-Fi 配置 |
| 网络是否通畅 | 确认 Wi-Fi 可以上网 |
| 密码是否变更 | 更新 Wi-Fi 密码 |

如果你是在“同步录音”时失败，应用会直接弹出 Wi-Fi 配置窗口，方便你立刻把当前网络重新写入徽章。

> **连接状态反映真实设备状态，不是缓存。** 你看到的就是当前实际情况。

## 常见问题

**Q: 没有录音徽章也能用 SmartSales 吗？**
可以。你可以手动导入手机上的录音文件，使用转写、分析和日程管理功能。

**Q: 一个手机可以配对多个徽章吗？**
目前支持单设备配对。

**Q: 配对后徽章需要一直靠近手机吗？**
录制时不需要靠近手机。录音完成后，徽章通过 Wi-Fi 独立上传。蓝牙连接主要用于状态监控和同步触发。

**Q: 徽章固件怎么更新？**
固件更新通过配对连接自动推送，会在连接时提示你。
