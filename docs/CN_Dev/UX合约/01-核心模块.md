## 1. 主屏幕 (Home Screen)

> **状态**: ✅ 已发布
> **概述**: 以聊天为中心的"高管办公桌"布局。关闭日程抽屉后显示的基础层。

### 布局结构

屏幕顶部是页眉，中部是基础层 (Home Hero)，底部是音频管理器触发区。

- **Header**: 包含历史抽屉触发器 `[☰]`，设备状态 `[📶]`，可编辑的会话标题，以及新建会话 `[➕]`。
- **Top Drawer (Scheduler)**: 位于上方，启动时自动下落。通过下拉把手展开。
- **Base Layer (Home Hero)**:
    - **Aura**: 呼吸光效，显示问候语 ("下午好, Frank")。
    - **Mode Toggle**: 切换 [ Coach | Analyst ] 模式。
    - **Input Area**: 输入框，带有附件按钮 `[📎]` 和 Shimmer 效果的提示语。
- **Right Toolbar**:
    - `[≣]`: 听悟菜单 (章节预览/高光)。
    - `[📦]`: Artifacts 抽屉 (PDF/CSV/生成物)。
- **Bottom Drawer (Audio)**: 位于底部，上拉展开。

### 交互详情

| 元素 | 动作 | 结果 |
|------|------|------|
| `[☰]` | 点击 | 打开历史抽屉 (左侧) |
| `[📶]` | 点击 | 打开连接弹窗 |
| `Title` | 点击 | 重命名当前会话 |
| `[➕]` | 点击 | 开始新会话 (清除上下文) |
| **模式切换** | 点击 | 仅设置意图 (`Coach`=紫, `Analyst`=蓝)。**不发生导航**。 |
| **输入框** | 点击/聚焦 | 导航至 **聊天界面** (继承选定模式) |

## 2. 聊天界面 (Chat Interface)

> **状态**: ✅ 已发布
> **概述**: 支持 Coach 和 Analyst 模式的双模聊天界面。

### 布局结构 (Coach 模式)

快速、对话式。无可见的思维轨迹。

- **Header**: 显示当前意图/话题。
- **Chat Area**:
    - **User Bubble**: 用户输入。
    - **AI Bubble**: AI 回复。支持富文本建议块。
    - **Analyst Suggestion**: 当 Coach 发现需要深度分析时，会追加建议块 "💡 建议切换到分析师模式"。
- **Mode Toggle**: 位于输入栏上方，悬浮切换 `[ 💬 Coach | 🔬 Analyst ]`。

### 逻辑说明

1. 解析用户意图。
2. 如需查询关联库 (Relevancy Library)，无 UI 反馈。
3. 直接生成回复。
4. 判断是否需要深度分析 → 追加建议块。
> **注意**: Coach 模式 **不显示** 思考框 (ThinkingBox)。思考框是 **Analyst 模式** 专用的。

## 3. 日程抽屉 (Scheduler Drawer)

> **状态**: ✅ 已发布
> **概述**: 包含日历和时间轴的顶部抽屉。从顶部边缘下拉打开。

### 布局结构

- **Month Carousel**: 年份/月份轮播。点击即时更新网格。
- **Calendar Grid**: 可折叠日历。
    - **Folded (默认)**: 仅显示当前周 (1行)。
    - **Expanded**: 显示完整月 (5行)。通过下拉把手展开。
    - **Object Permanence**: 展开时，当前周位置固定，其他行在上方/下方显现。
- **Timeline**: 自适应堆叠的时间轴。
    - **Task Card**: `☐` 任务卡片。
    - **Inspiration Card**: `💡` 灵感卡片 (带有 `[问AI]` 按钮)。
    - **Conflict Card**: `⚠️` 冲突卡片 (带有 `[展开]` 按钮)。

### 卡片交互

| 动作 | 触发器 | 结果 |
|------|--------|------|
| **左滑** | 任意卡片 | 立即删除 |
| **点击主体** | 任务/冲突 | 展开至聊天界面 |
| **点击 [问AI]** | 灵感卡片 | 进入 **多选模式** |

### 多选流程 (灵感卡片)

1. 用户点击卡片A上的 `[问AI]`。
2. 进入多选模式，卡片A自动选中。
3. 用户可点击其他卡片进行多选。底部显示 `[ 问AI (N) ]`。
4. 点击底部按钮 → 打开 Coach，将所有选中的卡片作为合并上下文。

## 4. 历史抽屉 (History Drawer)

> **状态**: ✅ 已发布
> **概述**: 左侧抽屉，用于会话历史导航。仅通过汉堡按钮打开 (无边缘滑动)。

### 组件详情

1.  **Device State Header (2行)**
    *   左: 电量 (Icon + %)
    *   右: 设备名 (WiFi Icon + SmartBadge)
    *   下: 状态文本 ("已连接 • 正常")
    *   **动作**: 点击打开连接弹窗。

2.  **Session List (分组)**
    *   `📌 置顶` (Pinned)
    *   `📅 今天` (Today)
    *   `🗓️ 最近30天` (Last 30 Days)
    *   `🗂️ YYYY-MM` (Month Archives)
    *   **格式**: `[客户名]_[摘要]` (如 `张总_Q4预算审查`)

3.  **User Footer**
    *   Avatar + Name + PRO Badge + Settings `[⚙️]`

### 交互规则

*   **打开**: 仅汉堡按钮 `[☰]`。
*   **关闭**: 点击遮罩 OR 左滑抽屉。
*   **长按会话**: 上下文菜单 (置顶, 重命名, 删除)。

## 5. 音频抽屉 (Audio Drawer)

> **状态**: ✅ 已发布
> **概述**: 底部抽屉，用于管理录音。从底部边缘上拉打开。

### 布局结构

- **Pill Handle**: 顶部把手。
- **Header**: `[ ↻ 同步中... ]` (打开时自动同步)。
- **List**:
    - **Row 1**: 星标 `[★]` + 名称 + 时间。
    - **Row 2**: 来源 `[☁]/[📱]` + 预览文本/转写引导。
    - **Local Audio**: 显示为 "右滑转写 >>>"。
    - **Action**: `[+] 上传本地音频`。

### 卡片状态

1.  **Collapsed (默认)**: 显示基本信息。
2.  **Expanded**: 仅当已转写时可用。
    *   显示完整日期、`[问AI]` 按钮。
    *   **Transcript Box**: 可折叠的转写内容预览。
    *   **Player Bar**: 进度条、播放控制。

### 手势

| 手势 | 动作 | 备注 |
|------|------|------|
| **右滑 →** | 立即转写 | 仅限未转写项。微光提示 "转写 >>>"。 |
| **左滑 ←** | 揭示菜单 | `[播放]` `[删除]` `[重命名]` |
| **点击主体** | 展开卡片 | 仅限已转写项。 |
| **点击 [问AI]** | 打开 Coach | 将转写内容作为上下文。 |

## 6. 用户中心 (User Center)

> **状态**: ✅ 已发布
> **概述**: 设置和个人资料管理面板。从历史抽屉底部进入。

### 包含板块

- **Profile Card**: 头像, 姓名, 职位, 等级, Plan Badge, 编辑入口。
- **Preferences**: 主题 (深/浅/跟随系统), AI 实验室 (记忆&学习), 通知设置。
- **Storage**: 已用空间, 清除缓存。
- **Security**: 修改密码, 生物识别, 登出所有设备。
- **Support**: 帮助中心, 联系与反馈。
- **About**: 版本信息, 隐私, 许可。
- **Logout**: 底部登出按钮。
