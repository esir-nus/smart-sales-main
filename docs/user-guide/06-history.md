# 06 历史记录

> 回到过去的对话 — 会话管理指南

---

## 什么是历史抽屉

历史抽屉存放你所有的对话会话记录。每次你和 AI 的对话都会自动保存，方便你随时回顾和继续。

**打开方式：**
- 点击左上角 **☰ 菜单按钮**
- 从屏幕 **左边缘向右滑动**

## 会话列表

### 分组规则

会话按时间自动分组：

| 分组 | 范围 |
|------|------|
| 📌 置顶 | 你主动置顶的会话 |
| 今天 | 今日的会话 |
| 昨天 | 昨日的会话 |
| 更早 | 再往前的会话 |

### 会话卡片

```
┌──────────────────────────────┐
│  帮我分析李经理的录音          │  ← 会话标题
│  4 月 16 日 · 8 条消息        │  ← 日期和消息数
└──────────────────────────────┘
```

- **标题**：默认为第一条消息的摘要，可手动重命名
- **日期**：最后活跃时间
- **消息数**：该会话中的消息总数

## 会话操作

### 加载会话

点击任意会话卡片，回到该次对话的完整上下文：
- 恢复所有消息历史
- 恢复绑定的录音上下文（如果有）
- 可以继续追问

### 长按菜单

长按会话卡片弹出操作菜单：

```
┌────────────────────────┐
│ 📌 置顶                 │
│ ✏️ 重命名               │
│ 🗑 删除                 │
└────────────────────────┘
```

#### 📌 置顶

- 将会话固定到列表顶部的「置顶」分组
- 适合正在进行的重要项目
- 再次长按可取消置顶

#### ✏️ 重命名

- 修改会话标题
- 用更有意义的名称替换自动生成的标题
- 例如：把"帮我整理录音"改为"张总项目-Q2 跟进"

#### 🗑 删除

- 删除该会话及其所有消息
- **删除前会请你确认**
- 删除后不可恢复

## 新建会话 vs 继续旧会话

| 场景 | 建议 |
|------|------|
| 新的客户、新的话题 | 点击 **+** 新建会话 |
| 同一客户的跟进讨论 | 从历史中加载之前的会话继续 |
| 不同录音的分析 | 新建会话（避免上下文混乱） |
| 接着上次没聊完的话题 | 从历史中加载继续 |

> **提示**：给重要的会话起个好名字（长按 → 重命名），以后找起来更方便。

## 常见问题

**Q: 会话数量有上限吗？**
没有严格限制，但过多的会话可能影响加载速度。建议定期清理不需要的旧会话。

**Q: 可以搜索历史会话吗？**
目前按时间分组浏览。给会话起有意义的名字可以帮助快速定位。

**Q: 删除会话会影响录音吗？**
不会。录音独立保存在录音抽屉中，删除会话只移除对话记录。

**Q: 切换到旧会话后，新的对话去哪了？**
每次新建或切换会话，之前的对话会自动保存。不会丢失。
