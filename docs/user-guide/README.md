## 关于 SmartSales

SmartSales 是一款面向销售人员的语音优先运营工具。它的核心理念是：

- **低摩擦采集**：用录音徽章捕获真实销售活动，不打断工作节奏
- **智能整理**：自动转写、摘要、提取关键信息，省去手动录入
- **行动驱动**：从录音回顾直接进入 AI 讨论和日程安排，让跟进不遗漏
- **人在回路**：AI 辅助而非自主行动，所有重要操作需你确认
