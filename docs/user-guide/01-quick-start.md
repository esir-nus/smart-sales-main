# 01 快速上手

> 从安装到进入主界面，5 分钟搞定

---

## 安装要求

- Android 8.0 及以上
- 需要的权限：麦克风、蓝牙、通知、存储
- 推荐搭配 SmartSales 录音徽章使用（非必需）

---

## 首次启动

第一次打开 SmartSales，你会进入 **引导设置流程**。整个过程分为 5 个步骤，按顺序完成即可。

### 步骤 1：欢迎页

```
┌─────────────────────────────────┐
│                                 │
│         ╔═══════════╗           │
│         ║ SmartSales ║          │
│         ╚═══════════╝           │
│                                 │
│    语音优先的销售运营助手         │
│                                 │
│                                 │
│                                 │
│      ┌─────────────────┐        │
│      │    开始使用       │        │
│      └─────────────────┘        │
│                                 │
└─────────────────────────────────┘
```

- 全屏深色界面，展示品牌标识
- 点击 **「开始使用」** 进入下一步

### 步骤 2：权限说明

```
┌─────────────────────────────────┐
│                                 │
│   SmartSales 需要以下权限：      │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 🎙 麦克风                │   │
│   │ 录制会议和语音交互        │   │
│   └─────────────────────────┘   │
│   ┌─────────────────────────┐   │
│   │ 📶 蓝牙                  │   │
│   │ 连接录音徽章              │   │
│   └─────────────────────────┘   │
│   ┌─────────────────────────┐   │
│   │ 🔔 通知                  │   │
│   │ 提醒日程和任务            │   │
│   └─────────────────────────┘   │
│                                 │
│      ┌─────────────────┐        │
│      │    继续           │        │
│      └─────────────────┘        │
│                                 │
└─────────────────────────────────┘
```

- 毛玻璃卡片逐项解释每个权限的用途
- 此时 **不会弹出系统权限弹窗**，只做说明
- 点击 **「继续」** 进入下一步

> **提示**：实际的系统权限请求会在你真正使用相关功能时才出现，减少初次使用的干扰。

### 步骤 3：语音握手

```
┌─────────────────────────────────┐
│                                 │
│      请说一句话，测试麦克风      │
│                                 │
│     ～～～～～～～～～～～～～    │
│      （实时音频波形显示）        │
│     ～～～～～～～～～～～～～    │
│                                 │
│                                 │
│      ┌─────────────────┐        │
│      │    继续           │        │
│      └─────────────────┘        │
│                                 │
└─────────────────────────────────┘
```

- 屏幕上显示实时音频波形
- 说一句话确认麦克风正常工作（约 1.5 秒即可确认）
- 点击 **「继续」**

### 步骤 4：唤醒录音徽章

```
┌─────────────────────────────────┐
│                                 │
│     请唤醒你的录音徽章           │
│                                 │
│   ┌─────────────────────────┐   │
│   │                         │   │
│   │    长按徽章按钮 3 秒     │   │
│   │    直到蓝灯开始闪烁      │   │
│   │                         │   │
│   └─────────────────────────┘   │
│                                 │
│      ┌─────────────────────┐    │
│      │ 蓝灯已经在闪了        │    │
│      └─────────────────────┘    │
│                                 │
└─────────────────────────────────┘
```

- 引导你长按录音徽章按钮 3 秒
- 等待设备上的蓝色指示灯开始闪烁
- 确认蓝灯闪烁后，点击 **「蓝灯已经在闪了」**

> **没有录音徽章？** 可以跳过此步骤，后续在「设置」中配对。你仍可以手动导入录音使用核心功能。

### 步骤 5：扫描配对

```
┌─────────────────────────────────┐
│                                 │
│       正在搜索附近设备...        │
│                                 │
│          ◌  ◌  ◌                │
│        ◌   📡   ◌              │
│          ◌  ◌  ◌                │
│      （雷达扫描动画）            │
│                                 │
│                                 │
│                                 │
│      ┌─────────────────┐        │
│      │    取消           │        │
│      └─────────────────┘        │
│                                 │
└─────────────────────────────────┘
```

- 应用通过蓝牙搜索附近的录音徽章
- 此时会请求蓝牙权限（首次在需要时请求）
- **不会自动连接** — 找到设备后需要你手动确认

找到设备后：

```
┌─────────────────────────────────┐
│                                 │
│       发现设备                   │
│                                 │
│   ┌─────────────────────────┐   │
│   │  SmartSales Badge       │   │
│   │  MAC: AA:BB:CC:DD:EE:FF │   │
│   │  信号: -42 dBm (强)     │   │
│   └─────────────────────────┘   │
│                                 │
│      ┌─────────────────┐        │
│      │    连接此设备     │        │
│      └─────────────────┘        │
│                                 │
└─────────────────────────────────┘
```

- 显示设备名称、MAC 地址和信号强度
- 点击 **「连接此设备」** 开始配对
- 配对过程中可能需要输入 Wi-Fi 信息（让徽章上传录音）
- 完成后自动进入主界面

## 主界面初印象

设置完成后，你会看到这样的界面：

```
┌─────────────────────────────────┐
│ ☰        [ 动态岛 ]         [+] │  ← 顶栏
│ 📶                               │  ← 设备徽章
├─────────────────────────────────┤
│                                 │
│      你好, SmartSales 用户       │
│      我是您的销售助手            │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
├─────────────────────────────────┤
│ [📎] 输入消息...          [发送] │  ← 输入框
└─────────────────────────────────┘
```

这就是你的工作基地。从这里，你可以：

| 操作 | 方法 |
|------|------|
| 和 AI 对话 | 直接在底部输入框打字 |
| 打开录音抽屉 | 从底部向上滑动 |
| 打开日程抽屉 | 点击顶部动态岛或从顶部向下拉 |
| 查看历史会话 | 点击左上角 ☰ |
| 新建对话 | 点击右上角 + |
| 查看设备状态 | 点击左侧设备徽章 📶 |
