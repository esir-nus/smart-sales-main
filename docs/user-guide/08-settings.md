## 偏好设置

### 主题切换

```
┌─────────────────────────────────┐
│  🎨 主题                        │
│                                 │
│  ○ 深色    ● 浅色    ○ 跟随系统 │
└─────────────────────────────────┘
```

| 选项 | 说明 |
|------|------|
| **深色** | 默认主题，深色背景，护眼且专业（推荐） |
| **浅色** | 浅色背景，适合户外或强光环境 |

- 设置会持久化保存

## 常见问题

**Q: 修改个人资料会影响之前的对话吗？**
不会。之前的对话保持原样，新的对话会使用更新后的信息。

**Q: 深色和浅色主题有功能差异吗？**
没有，只是视觉风格不同。

**Q: 清除缓存后加载会变慢吗？**
首次加载会稍慢，之后恢复正常。

**Q: 退出登录后数据安全吗？**
本地数据保留在设备上，不会被删除。如需彻底清除数据，请在退出前清除缓存并手动删除录音。
