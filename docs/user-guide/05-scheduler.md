# 05 日程管理

> 安排、追踪、完成 — 你的任务不再遗漏

---

## 什么是日程抽屉

日程抽屉是你管理所有待办任务的地方。它从屏幕顶部滑下，包含日历视图、任务时间线和冲突提醒。

**打开方式：**
- 点击顶部 **动态岛**
- 从动态岛 **向下拉** 手势

## 核心组件

### 日历区域

```
┌─────────────────────────────────┐
│     ◀  2026 年 4 月  ▶          │
│                                 │
│  一   二   三   四   五   六  日 │
│  14   15  [16]  17   18  19  20 │
│  ·         ·              ·     │
└─────────────────────────────────┘
```

- **月份导航**：◀ ▶ 切换月份
- **今天高亮**：当前日期用方框标识
- **任务标记**：有任务的日期下方显示小圆点
- 点击某天查看该日的任务列表

### 时间线

时间线按时间顺序排列当天的任务，包含三种卡片：

#### 任务卡片

```
┌─────────────────────────────────┐
│ ☐ 发送修改合同给张总             │
│   10:00  |  L3 普通              │
└─────────────────────────────────┘
```

- **☐ 复选框**：点击标记任务完成
- **任务名称**：简短的任务描述
- **时间**：计划执行时间
- **紧急度**：任务的优先级

**紧急度等级：**

| 等级 | 标识 | 含义 |
|------|------|------|
| L1 关键 | 🔴🔴 | 必须立即处理 |
| L2 重要 | 🔴 | 今日需要完成 |
| L3 普通 | （无标记） | 按计划执行 |
| 已关闭 | 灰色 | 已完成或取消 |

#### 冲突卡片

```
┌─────────────────────────────────┐
│ ⚠ 时间冲突                      │
│ 14:00 拜访李经理                │
│ 14:00 团队周会                  │
│ 两个安排在同一时间               │
└─────────────────────────────────┘
```

- 用醒目的警告样式显示
- 明确列出冲突的任务
- 不会静默忽略冲突 — 你需要手动调整

#### 灵感架

```
┌─────────────────────────────────┐
│ 💡 灵感架                        │
│ 准备季度报告的思路               │
└─────────────────────────────────┘
```

- 位于时间线底部
- 存放不需要具体时间的想法和笔记
- 轻量级的便签功能

## 任务操作

### 创建任务

在对话中让 AI 帮你创建：

```
你: 帮我安排明天上午 10 点发合同给张总

AI: 好的，我帮你创建一个任务：
    
    📋 发送合同给张总
    📅 明天 10:00
    🔔 优先级: 普通
    
    确认创建吗？

你: 好的

AI: ✓ 任务已创建，可以在日程抽屉中查看。
```

> **AI 不会静默创建任务** — 总是会展示内容并等你确认。

### 完成任务

```
操作前：
┌─────────────────────┐
│ ☐ 发送合同给张总     │
└─────────────────────┘

点击复选框后：
┌─────────────────────┐
│ ☑ 发送合同给张总     │  ← 划线 + 变灰
└─────────────────────┘
```

- 点击 **☐ 复选框** 标记完成
- 完成的任务显示为灰色删除线
- 完成后可能触发后续跟进提示（见下文）

### 删除任务

```
┌─────────────────────┐
│ 任务名称             │  ← 左滑
└─────────────────────┘
         ↓
    [确认删除？]
```

- **左滑** 任务卡片
- 确认后删除

### 调整时间

在对话中告诉 AI：

```
你: 把拜访李经理改到下午 3 点

AI: 好的，将"拜访李经理"从 14:00 调整到 15:00。
    
    调整后 14:00 的冲突也会消除。确认吗？

你: 确认

AI: ✓ 已调整。
```

## 冲突处理

SmartSales 会主动检测日程冲突，并通过多个渠道提醒你：

### 在动态岛中

```
┌─────────────────────────────────┐
│ ☰  📶  [⚠ 14:00 有两个安排冲突] [+] │
└─────────────────────────────────┘
```

### 在日程抽屉中

冲突卡片以醒目的样式展示在冲突的时间段旁。

### 处理方式

1. **调整时间**：在对话中或手动修改某个任务的时间
2. **取消任务**：删除优先级较低的任务
3. **保持冲突**：如果你知道两件事可以兼顾，可以忽略

> **SmartSales 不会自动解决冲突** — 所有调整需要你确认。冲突会一直显示直到你主动处理。

## 徽章完成后的跟进

当你通过录音徽章完成一次录音（比如客户拜访结束）后，系统可以创建一个跟进提示：

```
主界面中出现提示：
┌─────────────────────────────────────┐
│ ☰  📶  [ 动态岛 ]              [+]  │
├─────────────────────────────────────┤
│                                     │
│  ┌────────────────────────────┐     │
│  │ 🎯 拜访张总已完成           │     │
│  │    需要安排跟进吗？         │     │
│  │    [开始跟进]              │     │
│  └────────────────────────────┘     │
│                                     │
```

- 以 **提示芯片** 形式出现在主界面
- 点击后进入任务范围的对话
- **不会强制打断** 你当前的操作 — 始终是提示优先

## 实操示例

### 示例：整理会后跟进

```
场景：刚结束与客户的会议

1. 打开日程抽屉（点击动态岛）
2. 查看今天的时间线
3. 回到对话，让 AI 根据录音安排跟进：

你: 根据刚才和张总的会议，帮我安排跟进任务

AI: 根据会议录音，我建议以下跟进安排：

    1. 📋 发送修改合同给张总
       📅 4/18（周五）10:00  |  L2 重要
    
    2. 📋 确认技术部交付排期  
       📅 4/17（周四）09:00  |  L3 普通
    
    3. 📋 准备售后服务补充方案
       📅 4/21（下周一）14:00  |  L3 普通

    确认创建这些任务吗？

你: 第一个改成明天下午，其他 OK

AI: 好的：
    1. 发送修改合同给张总 → 4/17（周四）14:00
    2-3 保持不变
    
    确认创建？

你: 确认

AI: ✓ 3 个任务已创建，可在日程抽屉中查看。
```

## 常见问题

**Q: 日程抽屉可以看之前的任务吗？**
可以。通过日历导航切换到之前的日期，查看历史任务。

**Q: 任务完成后会消失吗？**
不会立即消失，会显示为灰色完成状态。

**Q: 可以创建重复任务吗？**
目前每个任务需要单独创建。你可以让 AI 批量创建。

**Q: 没有录音也能用日程功能吗？**
可以。日程功能独立于录音，直接在对话中告诉 AI 即可创建任务。
